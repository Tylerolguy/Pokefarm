# Pokefarm

Starter MonoGame project for a Pokemon-inspired farm and dungeon game.

## Current Prototype

- Large scrolling farm map
- Camera that follows the player
- Blue player cube
- WASD or arrow-key movement
- Simple farm boundaries at the edge of the map

## Requirements

- .NET 9 SDK or newer

## Run

After installing the .NET SDK:

```powershell
dotnet restore src/Pokefarm.Game/Pokefarm.Game.csproj
dotnet run --project src/Pokefarm.Game/Pokefarm.Game.csproj
```
